# Netflix Data Analysis

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset from Kaggle. The objective is to understand the data, perform cleaning, explore trends and patterns, create visualizations, and derive meaningful business insights.

## Dataset

* Dataset: Netflix Movies and TV Shows
* Source: Kaggle

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Project Steps

### Step 1: Load and Inspect the Data

* Loaded the dataset using Pandas.
* Checked shape, data types, missing values, and duplicate records.
* Summarized the dataset.

### Step 2: Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Converted date columns to datetime format.
* Created additional columns for analysis.

### Step 3: Exploratory Data Analysis

Answered the following questions:

1. What is the distribution of Movies and TV Shows on Netflix?
2. Which countries contribute the most content?
3. Which content ratings are most common?
4. How has content release changed over the years?
5. Which genres are most popular?

### Step 4: Data Visualization

Created the following charts:

* Bar Chart
* Line Chart
* Histogram
* Scatter Plot
* Pie Chart
* Heatmap

### Step 5: Insights Report

Generated five business insights based on the analysis.

## Key Findings

* Movies outnumber TV Shows on Netflix.
* The United States contributes the highest number of titles.
* Netflix experienced rapid growth in content additions after 2015.
* A few genres dominate the platform.
* Recent content is more prevalent than older content.
